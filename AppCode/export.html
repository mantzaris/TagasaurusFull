1) MEDIA FILES: Export all of the media content files in Tagasaurus. This means all of the image / audio / video / pdf files you have saved. A folder containing all of the files will be produced. Useful to back up your data and to transfer to another computer.
2) ANNOTATION INFORMATION (database file): Export all of the annotation information which includes the descriptions, emotions, memes, collections etc. This produces a clone of your database in a single database file which can be used to back up your data or transfer to another computer. (sqlite db file)
3) ND-JSON ANNOTATION INFORMATION: Newline delimited JSONs for all of the annotation information contained in the Tagasaurus database. A separate file for each database table where each entry in a table produces an independent JSON for that record. These ND-JSON files are placed in a separate folder in the export. (http://ndjson.org/)
