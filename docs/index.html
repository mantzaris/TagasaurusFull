power to the users

features of the application

Photos, Videos and Audio can be precious and that is why you need to keep them organized in one place on your computer to store. You can enjoy them at any time in pleasant viewing that contains your descriptions of them. Your thoughts and feelings can be stored to remember them in the future.

Media

store all your photos / videos / audio to view them and search for them easily with simple interfaces that uses machine learning

You can keep all your precious memories on your own computer. The database can easily store millions of records and your thoughts about the different files for you to find them easily in the future. This database will belong to the user and takes input only from the user.

Database

Users have their own database capable of storing millions of records and information related to the media

Finding the photo or video you are looking for is not easy, and a user can spend a lot of time looking through files for a long time in search. Tagasaurus offers a unique approach! The descriptions and emotions can be used, such as I want a 'happy photo at the park'. Tagasaurus takes memes seriously and all media can have memes connected to them, so you can search for images which have memes of a certain type 'beach pics with a cat meme', or search for images associated with a meme 'all memes which are connected to a beach in rank' or even the memes such as 'which memes are ranked high for beach pics?'. Users can search for similarity of the faces on the images

Search

Superb search for your files based upon descriptions, emotions, memes, faces, collections, and more

Often pictures, videos and songs belong together instead of as individual files to search for (find one and you have the group). Tagasaurus lets a user make collections to store and search for them.

Collections

Users can make collections to group media content together and view in gallery

Machine learning can be used to place the emotions from faces in the pictures and videos directly which saves time. There is also facial recognition to find pictures and videos containing people from other pictures and videos making the searching easier if a user forgot the keywords/emotions needed for the search.

Machine Learning

Users can use machine learning to place emotions on images automatically, or search for images with similar faces

Just load a new image with faces into 'super search' and see what similar looking people can be found!

Facial Recognition

Tagasaurus provides facial recognition to search for images

You can download the executable for Windows or Linux which installs it on a user system, or a just use the executable to run the program. The executable option allows a user to easily copy and paste all the files (including database and files) to another computer of the same OS and just run. If a user copies the files and database without the executable to another operating system the executable for that operating system can be used on the database.

Install

Install on a computer or just run without installation using the executable

The database is completely cross platform and all the media files are stored in one place. You can back up all the content by copying the folder to put on another computer or on an external storage such as a large USB or external SSD. You can copy this data to another computer and use an executable for that computer as well making the preservation of your files simple.

Copy Database

Copy the Database and files together to place on another computer

A user can 'export' their data that makes a complete copy of all the information they provided (descriptions, meme choices, collections etc) and all the media files. The user can 'import' this back up to restore their information, and if changes/additions/deletions have been made then a 'merge' operations is performed.

Back Up & Merge

Back up your data by exporting to re-import later

Many users have large capacity external Hard Disks and USBs which can have Tagasaurus placed on them, so they can run them with all the data and media stored on it to take wherever they go. The external SSD is a great choice. Windows users can simply place it on the external media and then run it with ease. Linux users will find that many drives are formated in 'vFAT' and permissions are not available to run an executable, so a bash script is provided for them in a folder next to the executable that re-mounts the drive with exec privileges so run it. So users can keep on the external HD an executable for Windows and Linux and depending on the OS being used, both will be using the same Tagasaurus database to work on storing memories.

USB or external HD

Run on a usb or external hard drive

Users can go to github.com/mantzaris/tagasaurus to find all the code there and check the download integrity using the hashes. Tagasaurus does not delete any content from a users computer or send data anywhere.

Open Source

All the code is open source (github.com/mantzaris/tagasaurus)

How...

Usage

  • The main menu lists the main actions to take and some scores to track the % of the images have been tagged or not.

  • New images can be loaded into the app when you go into the tagging page.

  • The tagging phase allows you to add text, emotions and memes (which are other images and any image can become a meme for another image). Search from images there.

  • Collection can be produced but in this page new images cannot be imported; only during tagging. A collection creation wizard helps a user produce a collection and users can search for collections.

  • Searching allows you to find the rank of relevant images and memes based upon the overlap of the annotations with the search criteria.

  • Super search allows a user to find images based upon facial recognition or emotion overlaps.

  • Export or import data. Importing allows data for the same media to be merged into a total description.

  • Install it on your computer, run the executable on your computer or even run it on a USB / external HD.
